# Słówka
    Do pliku "words.txt"  wpisujemy słówka pierwsze to słowo do nauczenia a drugie to tłumazenie, słowa odzielamy średnikiem. Zamiast pojedyńczych złów mogą być też zwroty.
    Każda para "słowo_do_nauczenia;tłumaczenie" w osobnej lini, czyli 5 słów to 5 lini.

    Przykład:
        cel:1
        harvest;żniwa/zbiór
        bounty;hojność
        growth;wzrost

    W takim układzie pojawi się pytanie podczas gry o znaczenie "żniwa/zbiór" i trzeba odnaleźć  "harvest". Z następnymi słowami jest tak samo.

    Jeśli ustawimy cel:2 (w pierwszej lini)  wtedy zostaniemy zapytani o znaczenie "harvest" i trzeba odnaleźć "żniwa/zbiór".
    W drugiej linii ustawiamy język czyli:
